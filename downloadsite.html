Download Site

This site is only is only a little site with few things to show. The highlight is that it is made of a RWD (Responsive Web Design). This means that the layout of this site scales automatically from phones to tablets to desktops.
To realize this project I used the "Twitter Bootstrap" Framework. Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Bootstrap, originally named Twitter Blueprint, was developed by Mark Otto and Jacob Thornton at Twitter as a framework to encourage consistency across internal tools. Before Bootstrap, various libraries were used for interface development, which led to inconsistencies and a high maintenance burden.

Bootstrap is open source. It's hosted, developed, and maintained on GitHub.

Source code available on GitHub.

Features

  • Responsive Web Design
  • Bootstrap
  • blueimp Gallery with the extension Bootstrap Image Gallery
  • Tile design
